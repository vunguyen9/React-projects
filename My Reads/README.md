# MyReads Book Shelf

This app can help you organize your books into categories of "currently reading", "want to read", and "read". You can also search more books to add it to your shelfs.

## React Version

- React 16.13.1

## Setup

To run this project install it using NPM:

- cd /myreads
- npm install
- npm start
